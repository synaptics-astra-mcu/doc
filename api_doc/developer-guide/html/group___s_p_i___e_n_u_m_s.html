Detailed Description

Enums for SPI driver.

This group defines various enumerations used by the SPI driver to configure and operate the SPI peripheral. These enums abstract hardware values and improve code readability.

  • spi_instance_en specifies the available SPI hardware instances on the platform.
  • spi_mode_en defines whether the SPI block is operating as a controller (master) or peripheral (slave).

These enums are used in SPI initialization, mode setting, and peripheral control.
