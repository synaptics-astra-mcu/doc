Detailed Description

Enums for Pinmux driver.

Represents the symbolic names for SoC ball pins that can be configured using the Pinmux driver. These names abstract the physical pin connections and allow developers to refer to pins functionally in firmware.

Each enumerator corresponds to a unique physical ball on the chip, associated with peripherals like I2C, SPI, JTAG, I2S, SDIO, GPIO, etc.
