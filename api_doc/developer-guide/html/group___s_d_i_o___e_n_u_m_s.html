Detailed Description

Enums for SDIO driver.

Contains enumerations used to manage and configure the SDIO (Secure Digital Input Output) interface in the system. These enums are used to abstract platform-specific identifiers and enable readable configuration code.

  • sdio_instance_en specifies the available SDIO hardware instances on the platform.

These enums assist in selecting the correct SDIO controller during initialization, data transfer operations, and power management.
