Detailed Description

Enums for I2C driver.

Contains enumerations used to configure and control the I2C (Inter-Integrated Circuit) interface in the system. These enums help in abstracting low-level configurations such as instance selection, speed modes, addressing modes, and operational flags.

  • i2c_instance_en specifies the available I2C hardware instances on the platform.

These enums assist in initializing the I2C controller, managing communication, and handling errors or specific operational states.
