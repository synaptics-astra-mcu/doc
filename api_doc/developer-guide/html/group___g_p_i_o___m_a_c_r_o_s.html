Detailed Description

Pin names for GPIO driver.

This group defines symbolic names for General-Purpose Input/Output (GPIO) pins. The macros map logical pin numbers to their corresponding bitmask values, which can be used for configuring, setting, or reading pin states.

The GPIOs are grouped as follows:

  • GPIO_0 to GPIO_31: Represent pins from GPIO Port 0.
  • GPIO_32 to GPIO_47: Represent pins from GPIO Port 1.
