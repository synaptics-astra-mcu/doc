Detailed Description

Enums for UART driver.

Provides all the necessary enumerations for configuring and using the UART peripheral.

Features:

  • Support for multiple UART instances (UART0, UART1, UARTLP)
  • Low-power UART support
  • Fully configurable baud rate, parity, and flow control
  • Interrupt and DMA modes
