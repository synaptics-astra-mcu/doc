Detailed Description

Enums for Clock driver.

Contains all enumeration types used within the Clock driver. These enums define key configuration options and identifiers for managing the clock system in a platform-agnostic, readable way. The enumerations provide abstraction over:

  • PLL Configuration Levels (clk_pll_config_en): Specifies different operational modes for PLLs such as off, bypassed, and frequency levels like low, mid, high.
  • Clock Enable Status (clk_enable_status_en): Indicates whether a specific clock is currently enabled or disabled.
  • PLL Identifiers (clk_pll_en): Identifies specific PLL blocks available on the SoC.
  • Clock Types (clk_type_en): Classifies clocks into types like oscillator, gate, PLL, or specific subsystem clocks like I2S or CGL.
  • Clock Identifiers (clk_ids_en): Enumerates all valid clock sources in the system and is used to reference specific clocks during configuration or status checks.

These enumerations are essential for configuring and managing the clock tree, allowing for modular control of power and performance-related settings.
